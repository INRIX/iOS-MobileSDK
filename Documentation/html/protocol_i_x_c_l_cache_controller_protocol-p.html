◆ cacheExpiryTimeInterval

- (NSTimeInterval) cacheExpiryTimeInterval
readwritenonatomicassign

Expiry time for the objects cached in the cache.

Discussion
Objects are expired after this time. The expired objects are not removed immediately at expiry but are eligible to removed if cacheLimit is reached and cache need to be trimmed. Defaults to 0 seconds. Subclasses must specify this.

◆ cacheLimit

- (NSUInteger) cacheLimit
readwritenonatomicassign

Size of the cache.

Discussion
This represents the maximum number of objects that can reside in the cache at given point of time. The default value is 50.
