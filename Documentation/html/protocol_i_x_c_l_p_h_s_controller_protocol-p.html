◆ anonymousVehicleId

- (NSString*) anonymousVehicleId
readnonatomiccopy

A string that uniquely identifies data sent by a vehicle/device without revealing its identity.

Discussion
An ID that reveals the identity of a vehicle, such as a Vehicle Identification Number (VIN), should not be used. AnonymousVehicleID may be changed as frequently as once an hour.
