INRIX Client Library for iOS

The <InrixCore> protocol is the primary interface for calling INRIX web services. It exposes a number of controller protocols that mediate communication between the client application and the INRIX services. The application creates a singleton InrixCore reference using the global IXCLCreateInrixCore() function and performs all INRIX-related tasks through the InrixCore controller properties.
