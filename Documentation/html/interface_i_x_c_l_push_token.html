Detailed Description

An IXCLPushToken provides INRIX with the necessary information to send Pre-Drive Alerts and other push notifications to a specific device. Once instantiated, send the IXCLPushToken to INRIX using IXCLAuthController's requestUpdatePushNotificationToken method.
