Parameters
timeIntervalThe number of seconds between firings of the timer. If timeInterval is less than or equal to 0.0, this method chooses the nonnegative value of 0.1 milliseconds instead.
nameA name for the timer for debugging and diagnostic purposes.
timerBlockThe block to call when the timer fires.
runsInBackgroundIf false the timer will be automatically paused when the app enters the background.
startImmediatelyIf true, the timer will begin countdown immediately. If false, the timer will not begin countdown until start is called.
Returns

- (void) pause

Pauses the running timer. A paused timer may be started or resumed.

Discussion
If timer is already paused or stopped, then calling this method has no effect.
