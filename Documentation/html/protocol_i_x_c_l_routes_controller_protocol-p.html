◆ maxRouteDistanceInMiles

- (double) maxRouteDistanceInMiles
readnonatomicassign

Maximum distance for requesting routes.

Discussion
Routes are not loaded from the server if the distance between the start point and end point is greater than this maximum distance. In order to request longer routes, set the overrideMaxDistance property to YES in the options object.
