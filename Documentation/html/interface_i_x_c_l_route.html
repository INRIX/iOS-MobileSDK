§ hasRestrictions

- (BOOL) hasRestrictions
readwritenonatomicassign

A Boolean value indicating if there are restrictions along the route.

Discussion
A restriction could be times when traffic is not allowed to flow in a particular direction. If YES, then travel times will not be returned; otherwise travel times are returned.

Discussion
