§ invokeService:queryParameters:completion:()

- (void) invokeService: (NSString *)  actionName
queryParameters: (NSDictionary *)  queryParameters
completion: (void(^)(NSString *result, IXCLCoreResponse *coreResponse))  completion 

Asynchronously invoke a CS service directly.

Discussion
This method allows the caller to invoke a CS service directly through a GET request. This method will automatically include the current access token. Results are returned through a completion block.
Parameters
actionNameCS Action to query.
queryParametersDictionary of NSString query parameters for the named action. This dictionary should not contain the current access token or the action name.
completionCompletion block that will be called with the server's response.

Returns the version of the INRIX Client Library.

The version consists of a major version, a minor version, a maintenance release number, and a build number. The build number consists of a date and a time. Example: 1.0.0.1406181422.

§ languageIdentifier

- (NSString*) languageIdentifier
readwritenonatomicassign

The language identifier to use for the INRIX web service API calls that support a language or locale parameter.

Discussion
This property defaults to the first language in the user's language preference setting. If this property is set to nil, no language parameter is passed to the INRIX web service calls. The identifier is expected to be a two character ISO 639-1 code (called a language designator by Apple) with an optional region code (called a region designator by Apple). For example, "en" for english or "en-US" for english with US as a region.

Defaults to [[NSBundle mainBundle] preferredLocalizations].firstObject.
