- (IXCLSDKConfig*) sdkConfig
readnonatomicstrong

Model object managing SDK configuration parameters.

Discussion
Property contains latest available configuration parameters. If config parameters are expired then new parameters will be requested from the server. IXCLSDKConfigurationParametersDidChangeNotification will be sent as soon as new config parameters received from the server.
