◆ pauseAlertingOnIncidents()

- (void) pauseAlertingOnIncidents

Pause alerting on incidents.

Discussion
Pausing allows the state of the controller to be maintained while preventing any notifications from being delivered. Does nothing if controller is not in the IXCLDriveTimeAlertsStateStarted state.

◆ resumeAlertingOnIncidents()

- (void) resumeAlertingOnIncidents

Resume alerting on incidents.

Discussion
The controller will resume notifications. Does nothing if controller is not in the IXCLDriveTimeAlertsStatePaused state.
