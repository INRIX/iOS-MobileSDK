◆ isStarted

- (BOOL) isStarted
readatomicassign

A Boolean value indicating whether regular location updates have been started.

Discussion
Location services are started by calling startUpdatingLocation. Location services are stopped by calling stopUpdatingLocation. The starting and stopping of regular location updates is affected by foreground/background state, location services permissions, and tripStatus.

◆ tripStatus

- (IXCLCurrentLocationManagerTripStatus) tripStatus
readnonatomicassign

An enumerated value that indicates the current status of a trip.

Discussion
The setting of this property affects the desiredAccuracy and starting or stopping of regular location updates. This setting is used as one indication of whether the use is driving.
